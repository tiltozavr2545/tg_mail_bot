# 📬 tg_mail_bot

Telegram-бот, который читает **новые непрочитанные письма** из Gmail и присылает тебе
их **краткую выжимку**, сделанную моделью **Gemini 2.5 Flash**.

Как только на почту приходит письмо — бот в течение интервала опроса видит его,
делает саммари и шлёт тебе в личку. Затем помечает письмо прочитанным.

## Как это работает

```
Gmail (IMAP) ──▶ читаем непрочитанные ──▶ Gemini 2.5 Flash ──▶ выжимка ──▶ Telegram
                          ▲                                                    │
                          └──────────── опрос раз в N секунд ──────────────────┘
```

- `config.py` — загрузка настроек из `.env`
- `mail_client.py` — IMAP: чтение непрочитанных, парсинг тела (в т.ч. HTML), пометка прочитанными
- `attachments.py` — конвертация вложений (PDF, DOCX, XLSX…) в Markdown через MarkItDown
- `summarizer.py` — вызов Gemini 2.5 Flash
- `main.py` — Telegram-бот (aiogram) + фоновый цикл опроса почты

## Вложения

Вложения писем автоматически конвертируются в текст через
[MarkItDown](https://github.com/microsoft/markitdown) и тоже попадают в выжимку,
так что Gemini «читает» и их содержимое. Поддерживаются **PDF, DOCX, PPTX, XLSX/XLS,
TXT, CSV, HTML** и другие текстовые форматы. Картинки и файлы тяжелее 15 МБ пропускаются.

## Что понадобится (4 секрета)

### 1. Токен Telegram-бота
1. Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → задай имя.
2. Скопируй токен вида `123456:ABC-DEF...` → это `TELEGRAM_BOT_TOKEN`.

### 2. Свой chat_id
1. Напиши [@userinfobot](https://t.me/userinfobot) — он пришлёт твой числовой `id`.
2. Это `TELEGRAM_CHAT_ID`. Бот будет слать выжимки только сюда.
3. **Важно:** один раз напиши своему боту `/start`, иначе Telegram не даст ему написать тебе первым.

### 3. Пароль приложения Gmail (App Password)
1. Включи двухфакторную аутентификацию: <https://myaccount.google.com/security>
2. Создай пароль приложения: <https://myaccount.google.com/apppasswords>
3. Скопируй 16-символьный пароль → это `GMAIL_APP_PASSWORD` (обычный пароль от аккаунта **не подойдёт**).
4. `GMAIL_ADDRESS` — твой адрес вида `you@gmail.com`.

### 4. Ключ Gemini API
1. Зайди в [Google AI Studio](https://aistudio.google.com/app/apikey) → создай API-ключ.
2. Скопируй → это `GEMINI_API_KEY`.

## Установка и запуск

```bash
# 1. Создай файл с секретами из шаблона и заполни его
cp .env.example .env
# затем открой .env и впиши свои значения

# 2. Установи зависимости (uv сам поднимет Python 3.12 и venv)
uv sync

# 3. Запусти бота
uv run python main.py
```

При старте бот проверит подключение к Gmail, пришлёт тебе «✅ Бот запущен»
и начнёт следить за почтой.

## Команды бота

- `/start` — статус и список команд
- `/check` — проверить почту прямо сейчас, не дожидаясь следующего опроса
- `/help` — помощь

## Настройки (в `.env`)

| Переменная | Назначение | По умолчанию |
|---|---|---|
| `POLL_INTERVAL_SECONDS` | Как часто проверять почту (сек) | `60` |
| `SUMMARY_LANGUAGE` | Язык выжимки | `русском` |
| `GEMINI_MODEL` | Модель Gemini | `gemini-2.5-flash` |
| `IMAP_HOST` | IMAP-хост | `imap.gmail.com` |

## Заметки

- Бот реагирует на команды только из твоего `TELEGRAM_CHAT_ID` — чужие сообщения игнорируются.
- Письма помечаются прочитанными **только после** успешной отправки выжимки (или уведомления об ошибке),
  поэтому ничего не теряется при сбоях.
- Секреты лежат в `.env`, который добавлен в `.gitignore` — он не попадёт в git.
